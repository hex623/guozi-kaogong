## 📝 待完善

### TabBar 图标优化
当前使用的是临时图标，建议使用以下方式获取更好的图标：

**方式1：使用微信小程序官方图标库**
在 `app.json` 中修改 tabBar 配置，使用系统图标：
```json
"tabBar": {
  "list": [
    {
      "pagePath": "pages/index/index",
      "text": "首页",
      "iconPath": "images/home.png",
      "selectedIconPath": "images/home-active.png"
    }
  ]
}
```

**方式2：在线生成图标**
1. 访问 https://www.iconfont.cn/
2. 搜索关键词：首页、添加、书本、列表、用户
3. 下载 PNG 格式，尺寸 48x48 像素
4. 普通状态：灰色 (#999999)
5. 选中状态：深色 (#333333)

**方式3：使用系统符号**
可以使用 SF Symbols 或系统 emoji 作为临时图标

## 📱 使用说明

### 设置考试
1. 进入「我的」页面
2. 点击「考试名称」输入考试类型（如：2024国考）
3. 点击「考试日期」使用滚轮选择日期
4. 返回首页即可看到带名称的倒计时

### 设置提醒
1. 开启「每日提醒」开关
2. 点击「提醒时间」使用滚轮选择时间
3. 到点会收到复习提醒（需开启订阅消息）
