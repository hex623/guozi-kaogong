# 🎯 郭子考公 - 智能错题本小程序

一款基于**艾宾浩斯遗忘曲线**的智能错题记录与复习工具，专为考公人群设计。

## ✨ 核心功能

### 📷 错题录入
- 支持相机拍摄或相册选择题目照片（最多3张）
- 文字或图片形式的正确答案
- 记录错误答案（便于分析错误原因）
- 题目出处标记（如"2024国考行测"）
- 灵活的自定义标签系统

### 🧠 智能复习（艾宾浩斯遗忘曲线）
- **自动推送**：第1/2/4/7/15天自动出现待复习题目
- **答题模式**：题目展示 → 查看答案 → 掌握度标记
- **自适应调整**：标记"还需复习"会增加推送频率
- **进度追踪**：每道题的复习历史和掌握状态

### 📊 数据统计
- 核心数据：总错题数 | 已掌握数 | 待复习数
- 标签分布可视化
- 近7天/30天学习趋势
- 连续打卡天数
- 易错分析TOP榜

### 🔔 辅助功能
- 每日打卡提醒（微信订阅消息）
- 考公倒计时
- 数据导出
- 错题分享（生成精美卡片）

## 🛠️ 技术架构

```
郭子考公
├── 前端：微信小程序原生框架
├── 后端：微信云开发
│   ├── 云数据库（questions, review_records）
│   ├── 云存储（图片存储）
│   └── 云函数（复习列表、状态更新）
└── 核心算法：艾宾浩斯遗忘曲线（1/2/4/7/15天）
```

## 📱 页面结构

| 页面 | 功能 |
|------|------|
| 首页 | Dashboard统计、快速录入入口 |
| 录入 | 拍照/选图、填写答案、打标签 |
| 复习 | 今日待复习列表、答题交互 |
| 题库 | 全部错题、标签/状态筛选 |
| 详情 | 错题详情、复习历史、删除 |

## 🚀 快速开始

### 1. 环境要求
- 微信开发者工具
- 开通微信云开发环境
- Node.js 14+

### 2. 安装运行

```bash
# 克隆项目
git clone https://github.com/yourusername/guozi-kaogong.git
cd guozi-kaogong

# 使用微信开发者工具打开项目
# 1. 导入项目，选择项目目录
# 2. 填写你的 AppID
# 3. 开启云开发

# 部署云函数
# 在微信开发者工具中：
# 1. 右键 cloudfunctions/getReviewList → 创建并部署：云端安装依赖
# 2. 右键 cloudfunctions/updateReviewStatus → 创建并部署：云端安装依赖

# 创建数据库集合
# 在云开发控制台数据库中创建：
# - questions（错题集合）
# - review_records（复习记录集合）
```

### 3. 数据库结构

**questions 集合：**
```json
{
  "photos": ["云存储文件ID数组"],
  "answerType": "text/image",
  "correctAnswer": "文字答案",
  "correctAnswerImage": "答案图片ID",
  "wrongAnswer": "错误答案",
  "source": "题目出处",
  "tags": ["标签数组"],
  "addDate": "添加日期",
  "reviewCount": 0,
  "status": "learning/mastered",
  "needsMoreReview": false,
  "wrongCount": 1
}
```

**review_records 集合：**
```json
{
  "questionId": "关联的题目ID",
  "date": "2024-02-18",
  "completed": true,
  "isMastered": true
}
```

## 📖 使用说明

### 第一次使用
1. 设置考试日期（我的页面）
2. 开启每日提醒（可选）
3. 录入第一道错题

### 日常学习流程
1. **录入**：拍照 → 填写答案 → 打标签 → 保存
2. **复习**：首页查看今日待复习 → 答题 → 标记掌握程度
3. **回顾**：定期查看错题库，分析易错标签

### 艾宾浩斯复习周期
- 第1次：第1天（初次录入后第二天）
- 第2次：第2天
- 第3次：第4天
- 第4次：第7天
- 第5次：第15天

完成5次复习后，题目自动标记为"已掌握"。
