# guozi-kaogong

微信小程序开发配置说明

## 云开发环境配置

### 1. 开通云开发
在微信开发者工具中：
1. 点击工具栏 "云开发" 按钮
2. 按提示开通云开发环境
3. 记录环境ID

### 2. 配置环境ID
修改 `app.js` 中的 env：
```javascript
wx.cloud.init({
  env: '你的环境ID',
  traceUser: true,
})
```

### 3. 创建数据库集合
在云开发控制台 → 数据库 → 创建集合：
- `questions` - 存储错题
- `review_records` - 存储复习记录

### 4. 部署云函数
右键点击云函数文件夹 → 创建并部署：云端安装依赖
- `getReviewList`
- `updateReviewStatus`

### 5. 配置权限
在云开发控制台 → 数据库 → 权限设置：
所有集合设置为 "仅创建者可读写"
